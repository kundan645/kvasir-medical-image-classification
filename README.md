# 🏥 Kvasir Medical Image Classification

Medical image classification on Kvasir GI tract dataset 
using Transfer Learning (MobileNetV2)

## 📊 Results
| Metric | Value |
|--------|-------|
| Validation Accuracy | **84%** |
| Best Class (normal-pylorus) | **96%** |
| Total Images | 8000 |
| Classes | 8 |

## 🛠️ Tech Stack
- Python, TensorFlow, Keras
- MobileNetV2 (Transfer Learning)
- Grad-CAM (Explainable AI)
- Google Colab (T4 GPU)

## 📁 Files
- `kvasir_classification.ipynb` — Main notebook
- `Kvasir_Project_Report.docx` — Project report
- `Kvasir_Presentation.pptx` — Presentation slides
- `kvasir_graphs.png` — Accuracy & Loss graphs
- `kvasir_report.csv` — Classification report

## 🚀 How to Run
1. Open `kvasir_classification.ipynb` in Google Colab
2. Download Kvasir dataset from Kaggle
3. Connect Google Drive
4. Run all cells in order
